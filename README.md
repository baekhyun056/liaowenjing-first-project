# liaowenjing-first-project
liaowenjing的第一个 GitHub 测试项目

## 项目介绍
这是 liaowenjing 创建的第一个 GitHub 测试项目，核心用途是熟悉 GitHub 仓库的创建、文件编辑、版本提交等基础操作，掌握 Git 版本控制的入门流程，为后续代码项目管理打下基础。

## 使用方法
本项目为 GitHub 入门练习项目，无需复杂配置，按以下步骤操作即可完成基础练习：
1. 在线浏览本仓库的 README.md 文件，了解项目结构与基础内容
2. 编辑 README.md 文件并提交修改，熟悉 GitHub 的文件更新与提交流程
